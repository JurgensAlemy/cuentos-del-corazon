# 💛 Cuentos del Corazón

Plataforma web educativa donde los estudiantes de Unidocencia de **Innova Schools** crean sus propios cuentos digitales para identificar, expresar y regular sus emociones — reemplazando el uso de herramientas externas como Book Creator.

🔗 **Demo en vivo:** [cuentos-del-corazon.vercel.app](https://cuentos-del-corazon.vercel.app)

## 📖 Sobre el proyecto

La propuesta **"Cuentos digitales que nacen del corazón"** integra el aprendizaje socioemocional (SEL), la inteligencia emocional de Daniel Goleman y el constructivismo, a través de la creación de cuentos digitales interactivos con texto, dibujos, imágenes y grabaciones de voz.

Más detalles sobre la fundamentación del proyecto están disponibles en la sección **"Acerca de"** dentro de la propia aplicación.

## ✨ Funcionalidades

- 🏠 **Inicio** — presentación del proyecto
- 🎭 **Explorador de Emociones** — historias y estrategias interactivas por emoción
- ✏️ **Crear mi cuento** — editor con texto, dibujo libre (canvas), subida de imágenes y grabación de voz
- 📚 **Galería** — cuentos publicados por los estudiantes (guardados localmente en el dispositivo)
- 📄 **Descarga en PDF** — cada cuento se puede exportar para imprimir o guardar
- ℹ️ **Acerca de** — fundamentación teórica y justificación del proyecto (para docentes y jurado)

## 🛠️ Stack técnico

| Tecnología | Uso |
|---|---|
| React + Vite | Framework y bundler |
| Tailwind CSS | Estilos |
| Framer Motion | Animaciones |
| React Router | Navegación entre páginas |
| Lucide React | Íconos |
| jsPDF + html2canvas | Generación de PDF del cuento |
| Vercel | Despliegue |

## 🚀 Cómo correr el proyecto localmente

```bash
npm install
npm run dev
```

El proyecto se abre en `http://localhost:5173`

Para generar el build de producción:

```bash
npm run build
```

## 📦 Despliegue

El proyecto está desplegado en Vercel y se actualiza automáticamente con cada `push` a la rama `main`.
